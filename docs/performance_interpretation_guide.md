# performance_tracking 분석 결과 해석 가이드

작성일: 2026-04-22
목적: tools/analyze_performance.py 출력 숫자를 객관 판단하기 위한
      사전 합의된 기준선.

## 1. 판단 원칙

규칙 1: 샘플 수 30 미만은 어떤 결론도 내리지 않는다.
  통계적 유의성 확보 불가. "데이터 부족, 3개월 이상 더 축적 후 재평가"로만 기록.

규칙 2: KOSPI 대비 초과수익으로만 판단한다.
  절대 수익률은 시장 베타일 뿐. 초과수익 = 우리 전략 수익 - 같은 기간 KOSPI 수익.

규칙 3: 단일 기간만 보지 않는다.
  1m 수익률이 좋아도 3m이 나쁘면 운 또는 과적합 의심. 최소 2개 기간 일관성 요구.

규칙 4: 한 번 정한 기준은 결과 보고 바꾸지 않는다.
  이 문서의 임계값은 데이터 보기 전에 확정. 사후 조정 금지.

## 2. 신호별 판단 기준 (KOSPI 대비 초과수익, 1개월)

| 신호        | 합격 (유지)   | 경계 (조정 검토) | 불합격 (재설계) |
|-------------|---------------|------------------|-----------------|
| strong_buy  | +3.0%p 이상   | 0 ~ +3.0%p       | 음수            |
| buy         | +1.5%p 이상   | 0 ~ +1.5%p       | 음수            |
| hold        | -1.0 ~ +1.0%p | ±1.0 ~ ±2.0%p    | 절댓값 2.0%p+   |
| sell        | -1.5%p 이하   | -1.5 ~ 0%p       | 양수            |

해석:
  - strong_buy가 시장을 3%p 이상 이겨야 "강력 매수" 신호로서 의미
  - buy는 1.5%p (강도 차이 반영)
  - hold는 시장 근처에 머물러야 정상. 크게 벗어나면 hold 판정 로직 잘못
  - sell이 양수면 "매도 신호가 오히려 오른 종목"이라 완전 반대

## 3. 점수 구간별 판단 기준 (KOSPI 대비 초과수익, 1개월)

| 점수 구간 | 최소 합격 초과수익 |
|-----------|--------------------|
| 90+       | +5.0%p             |
| 80-89     | +3.0%p             |
| 70-79     | +1.5%p             |
| 60-69     | 0 ~ +1.0%p         |
| 60 미만   | 기대 없음 (0 주변) |

점수와 수익률의 단조 증가 관계가 안 나오면 스코어링 로직 문제.
  예: 80점대가 90점대보다 수익 높으면 구조적 결함.

## 4. 적정주가 괴리율 대비 실제 수익률

fair_value_gap (현재가 vs 적정주가 하단):
  - 음수 = 저평가, 양수 = 고평가

기대 패턴:
  gap -30% 이하 (극저평가)   → 최고 수익률
  gap -30 ~ -10% (저평가)    → 중상위 수익률
  gap -10 ~ 0% (적정가 근처) → 중간
  gap 0 ~ +10%               → 중하위
  gap +10% 이상 (고평가)     → 최저 또는 음수

만약 이 패턴이 안 나오면:
  - 적정주가 모델(3-모델 가중평균)이 실제 시장과 괴리
  - 섹터 평균 PER·PBR·EV/EBITDA 값 재검토 필요
  - 업종 분류 오류 가능성

## 5. 월별 타임시리즈 일관성

| 관찰                                  | 해석                      |
|---------------------------------------|---------------------------|
| 초과수익이 꾸준히 양수                | 건강                      |
| 초과수익이 특정 1~2개월에 집중        | 운. 반복성 의심           |
| 초과수익이 불장에만 나오고 하락장 음수 | 모멘텀 편향. Value 보강 필요 |
| 초과수익이 하락장에만 나오고 불장 음수 | 역발상만 통함. 제한적      |
| 변동성이 KOSPI의 2배 이상             | 리스크 대비 보상 불충분    |

Sharpe ratio (연환산):
  - 1.0+ : 우수
  - 0.5~1.0 : 양호
  - 0~0.5 : 약함
  - 음수 : 전략 실패

## 6. 결정 플로우차트

```
분석 결과 나옴
  │
  ├─ 전체 샘플 수 30 미만?
  │    → 결론 유보. 3개월 추가 관찰
  │
  ├─ strong_buy 초과수익 +3%p 이상 AND 샘플 30+?
  │    → 전략 유효. 현재 로직 유지 + 백테스트로 확장 검증
  │
  ├─ strong_buy 초과수익 0~3%p?
  │    → 경계. 점수 구간별 / 월별 세부 분석으로 원인 파악
  │    → Magic Formula / F-Score 병행 스크리너 병행 검토
  │
  ├─ strong_buy 초과수익 음수?
  │    → 전략 재설계 필요.
  │    → 가중치 문제인지, 필터 문제인지, 시장 국면 문제인지 구분
  │    → 즉시 신규 투자 중단 권장 (실전 중이라면)
  │
  └─ 점수-수익률 단조성 깨짐?
       → 스코어링 로직 구조적 문제
       → 팩터 중요도 분석 후 가중치 재조정
```

## 7. 이 가이드를 언제 수정할 수 있나

원칙: 데이터 수집 3개월마다 한 번만.
이유: 결과 본 직후 기준 바꾸면 자기합리화.

수정 가능 조건:
  - 최초 3개월 축적 후
  - 시장 구조적 변화 (예: 세제 개편, 거래소 제도 변경)
  - 스코어링 로직 메이저 변경 시
